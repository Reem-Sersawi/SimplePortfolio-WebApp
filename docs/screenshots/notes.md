
## نظرة عامة
هذا المشروع يهدف إلى تحويل موقع البروفايل الشخصي إلى تطبيق معبأ في Docker ونشره بشكل احترافي على GitHub.

##  تحديات Docker وحلولها

### التحدي 1: اختيار صورة الأساس المناسبة
**المشكلة**: 
- استخدام صورة كبيرة الحجم (node:latest ~ 1GB)
- صعوبة في تحسين الأداء

**الحل**:
- استخدام `nginx:alpine` (فقط ~20MB)
- فصل بيئة البناء عن بيئة التشغيل
- استخدام ملفات ثابتة بسيطة

### التحدي 2: تهيئة Nginx للتطبيق
**المشكلة**:
- التطبيق يعتمد على Single Page Application (SPA)
- مسارات الروابط لا تعمل مباشرة مع Nginx

**الحل**:
- تعديل `nginx.conf`:
  ```nginx
  location / {
      try_files $uri $uri/ /index.html;
  }


### أهم درس من Git/GitHub:
تعلمت أهمية كتابة رسائل commit واضحة ومنظمة، واستخدام الفروع وPull Requests لتتبع التغييرات بشكل احترافي.
